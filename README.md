# Zenbit FullStack Tech Task

Welcome to Zenbit FullStack Tech Task repository!
This is an educational repository that uses the following technologies:
Frontend:
-react
-redux toolkit/RTK query
-react router dom
-styled components
-responsive markup
Backend:
-node.js
-mongoBD/mongoose
